Справочник по тегам и атрибутам HTML

Тег: < !DOCTYPE html<-нужен для указания типа текущего документа.

Тег: <pre>-нужен для определение блока предварительно форматированного текста.

Тег: <html>-нужен для заключение в себе все содержимое веб-страницы

Атрибуты - это дающие дополнительные возможности форматирования текста.

Атрибут тега <html> lang="en" - это контейнер, который заключает в себе все содержимое веб-страницы, включая теги head и body.

Одинарный тег - они не могут содержать текста или других тегов между открывающим и закрывающим тегами.

Двойной(закрывающий) тег - это...

Тег: <head> - нужен для хранения других элементов, цель которых — помочь браузеру в работе с данными.

Вложенный тег <meta> - нужен для хранения информации предназначенной для браузеров и поисковых систем.

Вложенный тег<title> - нужен для выполнения множество задач, как прямо, так и косвенно.

Тег: <Header> - нужен для задания «шапку» сайта или раздела, в которой обычно располагается заголовок.

Вложенный тег <P> - нужен для определения текстовых абзацов.

Тег: <footer> - нужен для задания «подвал» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
